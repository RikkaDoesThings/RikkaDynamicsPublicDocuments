# Privacy Policy

## 1. Introduction
This Privacy Policy explains how RikkaDynamicsUtilities handles data that may be collected, logged, or stored while the bot is used in Discord servers.

## 2. Information Collected
RikkaDynamicsUtilities may collect or store the following types of information:
- Message content for deleted or edited message logging.
- User IDs, usernames, and discriminator tags when logging moderation or ticket events.
- Guild IDs, channel IDs, and related metadata for server-specific configuration.
- Audit log entries when available and when permission is granted to infer deletion executors.

## 3. How Data is Used
Collected data is used to:
- provide moderation logs and ticket transcript history
- report activity and analytics
- support server configuration and settings
- infer message delete actions when message content is no longer available

## 4. Data Storage
- Data is stored locally or in the configured host environment, depending on how the bot is deployed.
- The bot does not automatically share logged data with third parties.
- Data retention is determined by the bot's configuration and any attached database or log storage strategy.

## 5. Bot Permissions
- The bot may request permissions needed for its features, such as message reading, logging, and audit log access.
- Users and servers should only grant permissions they are comfortable with.
- The bot does not require unnecessary permissions to function, but some features may need more access.

## 6. User Control
- Server owners can configure which features are enabled and which channels receive logs.
- If you do not want certain data logged, review the bot settings or disable the corresponding feature if available.

## 7. Security
- The bot owner and maintainers will take reasonable steps to protect stored data.
- Because the bot stores data in the hosting environment, security depends on the deployment setup and access controls.

## 8. Changes to Privacy Policy
This Privacy Policy can be updated when the bot changes or new features are added. Continued use of the bot after updates means you accept the revised policy.

## 9. Contact
For questions about privacy or data handling, contact the bot owner via creating a ticket in https://dsc.gg/rikkadynamics.
